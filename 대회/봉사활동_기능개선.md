# 봉사활동 기능 개선 내역

> 1365 자원봉사포털 API 데이터를 최대한 활용하여 봉사활동 탐색/신청 경험을 대폭 개선

---

## 신규 추가 기능 (7가지)

### 1. 신청 현황 실시간 표시
- **사용 데이터**: `appTotal` (현재 신청자), `rcritNmpr` (모집 정원)
- 프로그레스 바로 시각화 ("1/3명")
- 80% 이상 마감 임박 시 빨간색 경고 표시
- 학생이 경쟁률을 바로 파악 가능

### 2. 활동 요일 시각화
- **사용 데이터**: `actWkdy` ("1111100" = 월~금)
- 월~일 원형 칩으로 표시, 활동일은 파란색 강조
- 내 시간표와 비교하여 참여 가능 요일 판단 가능

### 3. 봉사 시간 자동 계산
- **사용 데이터**: `actBeginTm`, `actEndTm`, `progrmBgnde`, `progrmEndde`, `actWkdy`
- 일일 봉사 시간 + 총 예상 봉사 시간 자동 계산
- 예: "일 3시간 / 총 예상 ~45시간"
- 생활기록부 봉사 시간 계획에 활용

### 4. 가족/단체 참여 뱃지
- **사용 데이터**: `familyPosblAt`, `grpPosblAt`
- "가족 참여 가능" (핑크), "단체 가능" (남색) 뱃지
- 학부모와 함께 할 수 있는 봉사, 동아리 단체 봉사 필터링

### 5. 다중 장소 지도 표시
- **사용 데이터**: `areaLalo1/2/3` (위도,경도), `areaAddress1/2/3`, `actPlace`, `postAdres`
- Apple Maps에 봉사 장소 마커 표시 (최대 3곳)
- 상세 주소 텍스트 + "Apple 지도에서 열기" 버튼
- 봉사 장소까지의 거리/경로 바로 확인

### 6. 담당자 바로 연락
- **사용 데이터**: `nanmmbyNmAdmn` (담당자명), `telno` (전화번호), `email` (이메일)
- 담당자명 표시 + 전화 버튼 (tel: 링크) + 이메일 버튼 (mailto: 링크)
- 봉사 문의를 앱에서 바로 가능

### 7. 1365 신청 페이지 바로 연결
- **사용 데이터**: `progrmRegistNo` (프로그램 고유 ID)
- "1365에서 신청하기" 버튼 → 해당 봉사 상세 페이지로 직접 이동
- URL 형식: `https://www.1365.go.kr/vols/P9210/partcptn/timeCptn.do?type=show&progrmRegistNo={ID}`
- API의 url 필드가 없어도 progrmRegistNo로 자동 생성

## 기존 기능 개선

### 봉사 분야 표시
- **사용 데이터**: `srvcClCode` (교육, 환경, 문화 등)
- 정보 행에 봉사 분야 표시

### 모집 상태 뱃지 개선
- 모집중(초록), 모집대기(회색), 모집완료(회색) 색상 구분
- 청소년 가능 여부 파란 뱃지

## 기술적 차별점

1. **1365 API의 31개 필드를 100% 활용** — 기존 앱들은 제목/기간/기관 정도만 표시
2. **좌표 데이터로 지도 연동** — areaLalo 필드를 Apple Maps에 직접 표시
3. **봉사 시간 자동 계산** — 활동 시간 × 요일 × 주수로 총 예상 시간 산출
4. **신청 현황 실시간 시각화** — 프로그레스 바 + 마감 임박 경고
5. **1365 상세 페이지 딥링크** — progrmRegistNo로 URL 자동 생성, 바로 신청 가능
6. **Firebase 프록시** — 1365 HTTP API → HTTPS 변환 (iOS ATS 대응) + 6시간 캐시
