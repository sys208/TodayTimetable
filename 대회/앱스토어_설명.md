# App Store 등록 정보

## 앱 이름
오늘시간표

## 부제목 (30자 이내)
AI 기반 시간표·급식·학사일정 통합 앱

## 프로모션 텍스트 (170자 이내)
시간표 변경도 실시간 감지! NEIS·컴시간 공공데이터로 시간표, 급식, 학사일정, 학원 지도까지. AI 수행평가, 진로 리포트, Apple Watch, 위젯 지원.

## 설명

오늘시간표는 전국 초·중·고등학교 학생을 위한 스마트 시간표 앱입니다.
교육 공공데이터(NEIS)와 컴시간 알리미를 결합하여, 가장 정확한 시간표를 제공합니다.

▶ 실시간 시간표
• 컴시간 알리미 연동으로 보강, 자습, 교사 변경 실시간 반영
• 변경된 수업 자동 감지 및 강조 표시
• 일간/주간 시간표 보기
• 선생님 이름 표시
• 전국 초·중·고등학교 지원 (NEIS 교육정보 개방포털)

▶ 급식
• 오늘의 급식 메뉴 (조식/중식/석식)
• 19종 알레르기 정보 표시
• AI가 그려주는 급식 이미지 미리보기 (Google Gemini)
• 칼로리 정보

▶ 학사일정
• 월간 캘린더로 학교 행사 한눈에
• 시험 D-Day 자동 계산
• iOS 캘린더에 시험 일정 자동 추가
• 내일 학사일정 푸시 알림

▶ 위젯 & 잠금화면
• 홈 화면 위젯 4종 (시간표, 급식, 학사일정, 올인원)
• 잠금화면 위젯 (현재 교시 표시)
• Live Activity / 다이나믹 아일랜드

▶ Apple Watch
• 손목에서 시간표와 급식 확인
• 수업 5분 전 알림

▶ 공유
• 카카오톡으로 시간표/급식 공유
• 이미지로 공유 (인스타그램 스토리 등)
• 시험 D-Day 공유

▶ 봉사활동 (1365 연동)
• 지역별 봉사 프로그램 검색
• 모집 상태, 일정, 기관 정보 확인
• 봉사활동 실적 조회

▶ AI 수행평가 도우미
• 수행평가 과제 사진 → AI 자동 분석
• 평가 기준, 준비 방법 AI 코칭
• 과목별 수행평가 D-Day 관리

▶ AI 진로 리포트
• 몇 가지 질문에 답하면 관심 과목과 희망 진로를 바탕으로 진로 방향 정리
• 직업/학과/활동 계획을 보고서 형태로 확인
• 수행평가와 연결할 탐구 주제 추천
• 결과 PDF 저장 및 공유

▶ 학원 지도
• 주변 학원/교습소를 지도에서 확인
• 분야, 교습과정, 정원, 수강료, 주소, 전화번호 표시
• 지도 위치 기준 재검색
• Apple 지도 열기 지원

▶ 더 많은 기능
• 수업 5분 전 알림
• 비 오는 날 등교/하교 우산 알림
• 시간표 배경화면 자동 생성
• Siri 연동 ("오늘 시간표 알려줘")
• 교시별 시간 자동 설정 (컴시간 데이터)
• 시간표 변경 알림 (전날 저녁 푸시)
• 사진으로 시간표 입력 (AI 인식)
