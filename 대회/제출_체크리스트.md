# 제8회 교육 공공데이터 AI활용대회 제출 체크리스트

> 제출 기간: 2026년 5월 1일 ~ 5월 31일

---

## PPT (15매)

- [ ] 표지: 작품명 + 팀명 + 앱 아이콘
- [ ] 1매: 활용 데이터 표 (NEIS + 1365 + 커리어넷 + 워크넷 + 대학정보 + K-MOOC + KEDI CSV)
- [ ] 2매: 문제 인식
- [ ] 3매: 서비스 개요
- [ ] 4매: 시간표 (변경 감지)
- [ ] 5매: 급식 (AI 이미지)
- [ ] 6매: 학사일정 (D-Day)
- [ ] 7매: 봉사활동 + 수행평가 AI + AI 진로 리포트
- [ ] 8매: 학원 지도 + 수강료/정원 시각화
- [ ] 9매: 멀티플랫폼 + 위젯
- [ ] 10매: AI 활용 상세
- [ ] 11매: 기술 아키텍처
- [ ] 12매: 컴시간 변경 감지 알고리즘
- [ ] 13매: 기존 서비스 차별점
- [ ] 14매: 기대효과
- [ ] 15매: 향후 계획 + 요약 + QR코드

### PPT 주의사항
- [ ] 팀명 외 **학교명/개인명/지역명 없음** 확인
- [ ] 상업용 무료 폰트 사용 (프리텐다드)
- [ ] 공공누리 출처 표시
- [ ] 커리어넷/워크넷/대학정보/K-MOOC/KEDI CSV 출처 표시
- [ ] AI 진로 리포트는 "합격 가능성"이 아니라 "진로 탐색/활동 계획"으로 표현
- [ ] 스크린샷 실기기 캡처
- [ ] pptx 형식, 15매 이내

## 앱 배포

### iOS
- [ ] `APIConfig.isTestFlight = false` (학교 제한 해제)
- [ ] DEVELOPMENT_TEAM 설정
- [ ] App Store Connect 앱 등록
- [ ] 앱 설명 + 키워드 입력
- [ ] 스크린샷 10장 (6.7인치 + 6.1인치)
- [ ] 앱 아이콘 확인 (라이트/다크/틴티드)
- [ ] TestFlight 배포 → 테스트 → 심사 제출
- [ ] 개인정보 처리방침 URL
- [ ] AI 진로 리포트/학교 비서 답변에 참고용 고지 문구 포함
- [ ] 학원 지도는 위치 권한 거부 시 학교 주소 기준 검색 fallback 확인
- [ ] 우산 알림은 실제 마지막 교시/급식 유무 기준으로 동작 확인

### macOS
- [ ] Mac App Store 등록 (또는 DMG 배포)
- [ ] 스크린샷

### Android (선택)
- [ ] Play Store 내부 테스트
